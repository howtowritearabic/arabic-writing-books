# Arabic Writing Book Series

A simple website that displays preview pages from an Arabic writing book series designed for beginners.

The series helps learners practice writing Arabic letters, words, and basic handwriting step by step.

## About the Series

This project currently contains **3 parts**:

- Part 1 – Learning the Arabic alphabet
- Part 2 – Practicing letter connections and simple words
- Part 3 – Writing common words and short phrases

Each part includes:
- Book cover
- 1–2 preview pages

The goal of the series is to make **learning Arabic writing easier for beginners**.

## Website

The project is hosted using **GitHub Pages** so anyone can view the preview pages online.
